## Setup Instructions

### Virtual Environment
A Python virtual environment has been created. To activate it:

**Windows (PowerShell):**
```powershell
.\venv\Scripts\Activate.ps1
```

### Git Repository
This is a Git repository. To push to GitHub:

1. Create a new repository on GitHub
2. Add the remote origin:
```bash
git remote add origin https://github.com/your-username/home-work-2.git
```

3. Commit and push:
```bash
git add .
git commit -m "Initial commit: home work 2 content"
git branch -M main
git push -u origin main
```

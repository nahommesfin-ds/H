# Homework 1 Redone

## Overview
This project contains health data ethics quiz questions and answers, including case studies and best practices for data anonymization and protection.

## Setup Instructions

### Virtual Environment
A Python virtual environment has been created. To activate it:

**Windows (PowerShell):**
```powershell
.\venv\Scripts\Activate.ps1
```

### Git Repository
This is a Git repository. To push to GitHub:

1. Create a new repository on GitHub
2. Add the remote origin:
```bash
git remote add origin https://github.com/your-username/homework-1-redone.git
```

3. Commit and push:
```bash
git add .
git commit -m "Initial commit: homework 1 content"
git branch -M main
git push -u origin main
```

## Contents
- `homework.md` - Quiz questions and answers on health data ethics
- `venv/` - Python virtual environment (not tracked in git)

- Data Anonymization
- Cases: Havasupai Tribe, Project Nightingale, Change Healthcare, eGFR Bias
